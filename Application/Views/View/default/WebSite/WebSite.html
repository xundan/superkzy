一、公司介绍

我们是超级矿资源信息平台，公司全称是山东迅单网络科技有限公司。我们一家专注于煤炭供应链管理的服务企业，是国内最大的煤炭互联网信息平台，拥有无忧快煤和超级矿资源（微信号：cjkzy000）两大煤炭公众号。在煤炭业界有较高的地位。公司旗下拥有全国23个省市煤炭、货车资源，覆盖范围超过30万煤炭、货车用户。现平台已发展至建立微信号40多个、微信群1500多个、付费VIP客户达三万多人规模，线上成交额超过一千万。我公司在山西、陕西等地开设了办事处，致力于为客户提供代购、代销服务。公司主要业务是互联网+煤炭信息化业务，通过平台为顾客提供煤炭供应、求购、找车等信息服务。
